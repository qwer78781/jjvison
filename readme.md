# vision project

## Install DHT11 sensor
```
git clone https://github.com/adafruit/Adafruit_Python_DHT.git
cd Adafruit_Python_DHT
sudo python setup.py install
cd examples
cd Adafruit_Python_DHT/examples
```
- run
```
python AdafruitDHT.py 11 4
```
